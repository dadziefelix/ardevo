ABOUT THIS DEMO

Ardevo is a real-time fraud detection tool built specifically for Ghana's mobile money ecosystem. When you press RUN ANALYSIS, the system simulates 100 MoMo transactions and evaluates each one against a set of fraud detection rules based on documented attack patterns in Ghana.

WHAT THE NUMBERS MEAN

Analyzed — Total transactions processed in this run.
Flagged — Transactions identified as suspicious. These would trigger an alert in a live environment.
High Risk — Transactions above the high risk threshold requiring immediate attention.
Flag Rate — Percentage flagged. A realistic MoMo fraud rate in Ghana typically falls between 5 and 15 percent.

WHAT THE TRANSACTION FEED SHOWS

Each card represents a flagged transaction. Red border = high risk, amber = medium risk. Each card shows the transaction ID, type, sender phone number, amount in GHS, and mobile network.

This is a simulated demo. In a live deployment, Ardevo connects directly to a fintech platform's transaction feed and monitors in real time.
